# Excursions Platform / Codecademy Project  

This is a solution to the **"Excursion" landing page project** on Codecademy.  

In this project, the goal was to create a **responsive promotional webpage** for a fictional mobile app called **Excursion**, which helps users record and share their travel experiences.
The design emphasizes **visual storytelling** using **video backgrounds** and **landscape imagery** to evoke a sense of adventure and exploration.

## Overview

The **Excursions Platform** is a simple, single-page landing website built to demonstrate skills in **HTML**, **CSS**, and **version control with Git and GitHub**.  
The site promotes an upcoming travel app by presenting:
- A **video section** with a looping background video.
- Informative sections describing the app’s benefits.
- Visual content using travel imagery.
- A consistent minimalist layout optimized for accessibility and performance.

---

## Project Goals

- Build a **visually engaging** landing page for a mobile app.  
- Use **semantic HTML5** structure and **ARIA attributes** for better accessibility.  
- Apply **responsive design principles** for mobile and desktop devices.  
- Optimize media elements (images, video) with proper attributes like `loading="lazy"`, `decoding="async"`, and `preload`.  
- Practice version control: create commits, manage branches, and publish via **GitHub Pages**.  

---

## Features

- **Video Section** — autoplaying, muted loop video introduces the Excursion app.  
- **Informational Sections** — clean typography and simple copy explaining the product.  
- **Responsive Layout** — optimized for various screen sizes with adaptive text and image scaling.  
- **Accessible Structure** — ARIA labels, proper heading hierarchy, and alt text for images.  
- **SEO Optimized Head** — meta tags for Open Graph, Twitter Card, and canonical URL.
